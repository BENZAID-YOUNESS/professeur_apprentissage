# SmartList Airbnb – v1.0

Petit outil simple pour générer des checklists Airbnb en 30 secondes.

Fonctions :
- Ajouter / supprimer des articles
- Charger des templates (arrivée, ménage, départ)
- Export CSV
- Historique automatique

Utilisation :

1. Installer Python (https://python.org)
2. Dézipper Smartlist_Airbnb_v1.zip
3. Ouvrir le dossier
4. Lancer :

python run.py

Aucune connexion Internet requise.
Tout fonctionne en local.
